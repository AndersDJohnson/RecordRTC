new StereoRecorder(mediaStream)

JavaScript standalone object for stereo audio recording.

StereoRecorder is a standalone class used by RecordRTC to bring audio-recording in chrome. It runs top over StereoAudioRecorder.
Parameters:
Name Type Description
mediaStream MediaStream MediaStream object fetched using getUserMedia API or generated using captureStreamUntilEnded or WebAudio API.
Source:
  • RecordRTC.js, line 1133
Example
var recorder = new StereoRecorder(MediaStream);
recorder.record();
recorder.stop(function(blob) {
    video.src = URL.createObjectURL(blob);
});
